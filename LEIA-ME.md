# PM Track — Gran Cursos Online

## ▶️ Como rodar

Abra o terminal **dentro desta pasta** (pm-track-app) e execute:

```
npm install
npm run dev
```

Acesse no navegador: **http://localhost:5000**

---

## ⚠️ Requisito

Node.js v18 ou superior instalado.
Baixe em: https://nodejs.org (versão LTS)

---

## 🔑 Acesso admin

URL: http://localhost:5000/#/admin  
Senha: pmtrack-admin-2026
